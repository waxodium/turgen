# Turgen Shell
Turgen Shell is a is non POSIX-Compliance shell designed for fast paced usage, made for **Unix GNU/Linux** platform.

> [!WARNING]
> Still working on it..
> Expect breaking changes!

## Install

### Build from source
```bash
git clone --depth 1 https://github.com/waxodium/turgen.git && cd turgen

# May need to export ~/.local/bin as PATH 
make install CC=gcc
```
